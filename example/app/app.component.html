Default configurations

kx-modals has a couple of default configurations that you can use when combined with a CSS framework:

  • Semantic UI (v2.2): use SEMANTIC2 as configuration.
  • Bootstrap (v3.3.7): use BOOTSTRAP3 as configuration.
  • Bootstrap (v4.0.0-beta): use BOOTSTRAP4 as configuration.
  • Foundation 6 (v6.4.2): use FOUNDATION6 as configuration.
